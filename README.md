# Markdown Viewer Application This project provides a simple, single-file HTML application to view Markdown content, specifically designed to render `input.md` using the `marked.js` library and styled with Tailwind CSS. ## Features - **Markdown to HTML Conversion**: Automatically fetches `input.md` and converts its content to HTML. - **Responsive Design**: Utilizes Tailwind CSS for a clean and responsive user interface across various devices. - **Easy to Deploy**: A single `index.html` file means easy deployment to any static web server. ## Setup To run this application, ensure you have the following files in the same directory: 1. `index.html` (provided) 2. `input.md` (your Markdown content) Simply open `index.html` in your web browser. The page will automatically fetch `input.md`, convert it, and display the rendered HTML. ## Technologies Used - **HTML5**: For the core structure of the web page. - **Tailwind CSS**: For utility-first styling and responsive design. - **Marked.js**: A powerful Markdown parser and compiler for JavaScript. ## Contributing Feel free to fork this project, open issues, or submit pull requests for any improvements or bug fixes. ## License This project is open-sourced under the MIT License.
